## Responsive Behavior

### Desktop (1440px+)
- Full three-column layout where applicable
- Context panel as persistent sidebar
- Expanded graph visualization

### Tablet (768px - 1439px)
- Two-column layout
- Context panel as collapsible drawer
- Simplified graph view

### Mobile (< 768px)
- Single column, card-based
- Context as bottom sheet
- Focus on slice list and basic management
- Full AI sessions defer to Claude app

## Accessibility

### Screen Reader Support

All interactive elements include:
- Descriptive `aria-label` attributes
- Proper heading hierarchy (`h1` â†’ `h6`)
- Live regions for dynamic updates
- Focus management for modals/drawers

### Keyboard Navigation

- All actions accessible via keyboard
- Visible focus indicators
- Skip links for main content
- Escape to close modals/panels

### Visual Accessibility

- WCAG 2.1 AA contrast ratios
- No information conveyed by color alone
- Scalable text (respects system font size)
- Reduced motion option
